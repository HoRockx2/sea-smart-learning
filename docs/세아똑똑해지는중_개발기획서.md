# 세아 똑똑해지는 중 — 개발 기획서

> 이 문서는 「세아 똑똑해지는 중 웹페이지 기획서」와 디자인 시안을 기반으로, 실제 개발에 필요한 기술 명세를 정리합니다.

---

## 1. 프로젝트 구조

### 1.1 파일 트리

```
세아 똑똑해지는 중/
├── index.html                ← 메인 페이지
├── css/
│   └── common.css            ← 공통 스타일 (디자인 시스템 전체)
├── js/
│   └── common.js             ← 공통 스크립트 (카드 렌더링, 필터, 유틸)
├── assets/
│   └── icons/                ← 주제별 아이콘 (선택, 이모지로 대체 가능)
├── pages/
│   ├── math-carrying.html    ← 예시: 받아올림 덧셈
│   └── ...                   ← AI로 생성되는 상세 페이지들
└── data/
    └── topics.json           ← 주제 목록 데이터
```

### 1.2 기술 스택

| 영역 | 기술 | 비고 |
|------|------|------|
| 마크업 | HTML5 | 시맨틱 태그 사용 |
| 스타일 | CSS3 | CSS 변수 기반 테마, 미디어쿼리 반응형 |
| 스크립트 | Vanilla JavaScript (ES6+) | 프레임워크 없음, 모듈화 |
| 폰트 | Google Fonts — Noto Sans KR | 400, 500, 700 웨이트 |
| 데이터 | JSON (정적 파일) | fetch()로 로딩 |
| 호스팅 | GitHub Pages 또는 Netlify | 정적 사이트, 서버 불필요 |

```html
  <header class="header">
    <!-- 로고 + 인사말 -->
  </header>

  <nav id="filter-bar" class="filter-bar">

  </nav>

  <main id="card-grid" class="card-grid">
    <!-- JS가 topics.json 기반으로 카드 생성 -->
  </main>

  <footer class="footer">
    <!-- 저작권 + 부모님 안내 -->
  </footer>

  <script src="js/common.js"></script>
  <script>
    loadTopics('card-grid');
    initCategoryFilter('filter-bar', 'card-grid');
  </script>
</body>
</html>
```

### 5.2 동작 흐름

1. 페이지 로드 → `fetch('data/topics.json')` 호출
2. JSON 파싱 → 각 항목으로 `createTopicCard()` 실행
3. 카드를 `#card-grid`에 삽입
4. JSON의 카테고리 목록에서 중복 제거 → 필터 탭 동적 생성
5. 필터 탭 클릭 → 해당 카테고리 카드만 표시 (CSS display 토글)
6. 카드 클릭 → `topic.page` 경로로 이동

### 6.2 상세 페이지 동작 흐름

```
페이지 로드
  ↓
[개념 배우기] 섹션 표시
  ↓ (스크롤 또는 버튼)
[예시 문제] 섹션 표시
  ↓ (스크롤 또는 버튼)
[연습 문제] 섹션 활성화
  ↓
사용자 입력 → "정답 확인" 클릭
  ↓
checkAllAnswers() 실행
  ├── 각 문제별 checkAnswer() → 개별 피드백 표시
  └── showResults() → 결과 섹션 표시
       ├── 점수 계산 (맞은 수 / 전체)
       ├── 캐릭터 메시지 분기 (3개 이상 맞음 → 칭찬 / 미만 → 격려)
       └── "다시 풀기" + "개념 다시 보기" 버튼 표시
               ↓
          resetProblems() → 입력 초기화, 문제 셔플(선택), 연습 섹션으로 스크롤
```
